Important FAQ

What is the DOM?

The DOM (Document Object Model) is a programming interface for web documents. It represents the structure of a document as a tree of objects, where each node is an element, attribute, or text from the HTML or XML document. JavaScript can be used to interact with and manipulate these nodes.

How is the DOM Different form HTML?

HTML is the markup language used to structure web content, while the DOM is an object-based representation of that HTML content in the browser. When a web page loads, the browser converts HTML into the DOM, making it accessible and modifiable using JavaScript.

What are DOM Elements and Nodes?

DOM elements represent HTML tags (like div, p, a), while DOM nodes are the general term for all items in the DOM tree, which includes elements, text, attributes, and comments. Every element is a node, but not every node is an element.

What is Event Delegation in DOM?

Event delegation is a technique in JavaScript where you add a single event listener to a parent element instead of adding listeners to individual child elements. The event listener can capture events from child elements due to event bubbling. This improves performance when dealing with large numbers of elements.
